PROJECT CONTEXT
I have a SQU Weather Station project using a DataTaker DT80W Series 4 data logger. The logger records data from 4 sensors. My current workflow is:

1. DataTaker DT80W logs sensor data into .DBD files by schedule.

3. The script downloads the DBD files:
   - DATA_A.DBD
   - DATA_B.DBD
   - DATA_C.DBD
   - DATA_D.DBD
4. The script converts them into CSV files using dump_dbd.exe.
5. Each sensor has its own CSV file:
   - AQT560_DATA.CSV
   - WS500_DATA.CSV
   - SMP10_DATA.CSV
   - DR30_DATA.CSV
6. The dashboard must read these CSV files and display the latest readings and historical data.

- displayName
- csvFile

3. GET /api/sensors/:sensorId/data

- from

- sensor
